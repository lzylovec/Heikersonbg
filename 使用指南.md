# 🎤 社恐翻译器使用指南

## 🚀 快速开始

1. **打开应用**：访问 http://localhost:8080
2. **允许麦克风权限**：浏览器会提示需要麦克风权限，请点击允许
3. **开始录音**：点击"开始录音"按钮或按空格键
4. **说话**：对着麦克风说话（建议在安静环境，说话时间10秒内）
5. **等待分析**：AI会自动识别语音并分析客套话
6. **查看结果**：屏幕会显示原始文本和翻译后的真实意图

## 🎯 功能特点

### 🧠 智能分析
- **客套话识别**：自动识别中文社交中的客套表达
- **真实意图翻译**：揭示话语背后的真实含义
- **社恐友好建议**：提供适合社恐人士的回应建议

### 🎨 用户界面
- **现代化设计**：渐变背景，毛玻璃效果
- **响应式布局**：适配手机和电脑屏幕
- **实时状态显示**：清晰显示当前处理状态

### ⚡ 快捷操作
- **空格键**：快速开始录音
- **ESC键**：取消当前录音
- **清除按钮**：清空当前结果

## 💡 使用技巧

### 🎤 录音建议
- 保持环境安静，避免背景噪音
- 说话清晰，语速适中
- 每次录音控制在10秒内
- 使用标准普通话效果更佳

### 🤔 分析示例

| 原话 | AI分析结果 |
|------|-----------|
| "改天我们一起吃饭吧" | 客套话 - 我暂时没有具体计划，但希望以后有机会再聚 |
| "你最近怎么样" | 客套话 - 礼貌性问候，并非真正想了解近况 |
| "有空常联系" | 客套话 - 希望你以后偶尔记得我 |
| "你的想法很有意思" | 客套话 - 我暂时没太在意，或者我不太同意 |
| "我考虑一下" | 客套话 - 我暂时不想答应/我需要更多时间思考 |

### 🎯 社恐回应建议
- 对于客套话，保持礼貌但不过分热情
- 使用简短、明确的回应
- 避免过度解读，保持自然态度
- 记住：不是所有客套话都需要深度回应

## 🔧 故障排除

### 🎤 录音问题
- **无法录音**：检查麦克风权限是否开启
- **识别失败**：确保使用中文，环境安静
- **超时错误**：说话时间不要太长

### 🌐 网络问题
- **AI分析失败**：检查网络连接
- **API错误**：确认API密钥是否正确配置

### 🖥️ 界面问题
- **页面加载慢**：刷新页面重试
- **按钮无响应**：检查浏览器控制台错误

## 📱 兼容性

- **浏览器**：Chrome、Safari、Firefox、Edge
- **设备**：支持麦克风的电脑和手机
- **语言**：中文语音识别和分析
